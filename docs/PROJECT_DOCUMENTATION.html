VAANI — Voice-First Financial Advisor

A bilingual (Hindi/English) voice-first financial advisor for Indian users. Combines voice interaction with expense tracking, FD/SIP management, tax intelligence, and freelancer income management.

Stack: Expo SDK 54, React Native 0.81, TypeScript (strict)

Database: SQLite (offline-first) + Supabase sync

Voice: expo-av (recording), expo-speech (TTS)

🌐 Web App

Stack: React 19, React Router v6

⚙️ Key Services

1. Chat Service — chatService.ts

AI Backend: MiniMax M2.7 (streaming API) with Hindi/English system prompt

Fallback: Demo mode with pre-written responses

Features:

  • Extracts expense intents: [EXPENSE:500:food:chai]
  • Budget alerts: [BUDGET_ALERT:food:85]
  • Strips think tags from AI responses
  • Streams tokens in real-time

2. Money Management — moneyManagementService.ts

Database: SQLite with 8 tables (expenses, budgets, savings_goals, fd_investments, sip_investments, crypto_holdings, gold_holdings, transactions)

Key Functions:

  • addExpense(amount, category, description)
  • getExpenses(startDate, endDate, category)
  • setBudget(category, monthlyLimit)
  • addFDInvestment(bank, amount, rate, tenureMonths)
  • addSIPInvestment(fundName, amount)
  • getPortfolioSummary()

3. Idle Money Detection — idleMoneyService.ts

Detects when bank balance exceeds emergency buffer + upcoming EMIs. Suggests liquid funds.

Formula: idle_amount = total_balance - (upcoming_emis + monthly_budget + emergency_buffer)

4. Tax Intelligence — taxIntelligenceService.ts

  • Section 80C: ₹1,50,000 limit monitoring
  • Tax Harvesting: Identifies holdings with losses to offset gains
  • Advance Tax: Calculates quarterly installments
  • TDS Detection: Alerts for payments ≥ ₹30,000

5. Freelancer OS — freelancerService.ts

  • Income Logging: "Infosys se 50000 aaye"
  • Client Tracker: Payment history, total income per client
  • GST Invoice: 18% GST calculation, downloadable
  • ITR Export: Data formatted for income tax return filing
  • Auto TDS: Flags payments ≥ ₹30,000 for 10% TDS
